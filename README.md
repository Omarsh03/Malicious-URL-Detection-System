# Malicious URL Detection System

## Overview
End-to-end machine learning pipeline for detecting malicious URLs using supervised learning techniques.

## Project Pipeline
- Data cleaning and preprocessing
- Feature engineering (URL length, special characters, domain patterns)
- Text vectorization using TF-IDF and Bag-of-Words
- Classification using Logistic Regression and Random Forest
- Model evaluation and performance analysis
